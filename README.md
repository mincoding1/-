# 임베디드 트랙 반 대항전 (알고리즘 배틀 규정)

### 대항전을 하는 이유
알고리즘 IM 기출 유사문제 및 비공개 A형 기출 유사문제를 <br>
조금 더 재미있게, Problem Solving 문제를 풀어 보는 시간을 갖자는 취지.

### 상품
소규모 상품이 존재합니다.
1등반 - 간식 보급박스 (대 - 10만원)<br/>
2등반 - 간식 보급박스 (중 - 5만원)<br/>
3등반 - 간식 보급박스 (소 - 3만원)<br/>
<br/>

### 유의 사항!
- 이벤트성이기에, 즐기는 마음으로 참여 쿠다사이. 
- 부정행위 금지, 정황 발견 시 반대항전 중단. 
  (정답을 친구에게 전달, 사이트 해킹 시도, 문제 블로그 인터넷 검색)
<br/>

### 순위 결정 방법
- 1 ~ 5위 중 가장 많은 팀을 보유한 반이 1등
- 두 번째로 많은 팀을 보유한 반이 2등
- 세 번째로 많은 팀을 보유한 반이 3등
- 동 순위가 존재한다면,
  동 순위가 발생한 반 중에, 
  어느 반이 상위 순위 팀을 보유하고 있는지에 따라 결정됨.
<br/>

### 순위 결정 예시
A, B, C 반이 있다고 가정한다.
- 만약 1위부터 5위까지 { **A**, B, B, **A**, C } 반이라면, A반이 1등이다.
- 만약 1위부터 5위까지 { A, **B**, **B**, **B**, C } 반이라면, B반이 1등이다.
- 만약 1위부터 5위까지 { B, **A**, C, **A**, C } 반이라면, A반이 1등이다.
- 만약 1위부터 5위까지 { B, B, **C**, **C**, **C** } 반이라면, C반이 1등이다.
- 만약 1위부터 5위까지 { A, **B**, **B**, C, C } 반이라면, B반이 1등이다.
